# Music-Store-Sales-Analysis (Interactive Dashboard Using SQL + Power BI)

## Project Objective
The objective was to identify revenue drivers, customer purchasing patterns, and year-over-year trends.

## Dataset Used
<a href= "https://www.kaggle.com/datasets/saswatsethda/music-store-database">Dataset</a>

## Business Questions
- How is revenue performing over time?
- Are revenue changes driven by order volume or customer spending?
- How concentrated is revenue among top customers?
- How does customer spending behavior change year-over-year?

## Key Metrics
- Total Revenue
- Total Customers
- Total Orders
- Average Order Value (AOV)
- Monthly & Yearly Sales Trends
- Purchase Frequency
